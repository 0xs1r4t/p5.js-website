description: >
  <p>Creates a canvas element on the web page.</p>

  <p><code>createCanvas()</code> creates the main drawing canvas for a sketch.
  It should

  only be called once at the beginning of <a
  href="/reference/p5/setup/">setup()</a>.

  Calling <code>createCanvas()</code> more than once causes unpredictable
  behavior.</p>

  <p>The first two parameters, <code>width</code> and <code>height</code>, are
  optional. They set the

  dimensions of the canvas and the values of the

  <a href="/reference/p5/width/">width</a> and <a
  href="/reference/p5/height/">height</a> system

  variables. For example, calling <code>createCanvas(900, 500)</code> creates a
  canvas

  that's 900Ã—500 pixels. By default, <code>width</code> and <code>height</code>
  are both 100.</p>

  <p>The third parameter is also optional. If either of the constants
  <code>P2D</code> or

  <code>WEBGL</code> is passed, as in <code>createCanvas(900, 500,
  WEBGL)</code>, then it will set

  the sketch's rendering mode. If an existing

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLCanvasElement/"
  target="_blank">HTMLCanvasElement</a>

  is passed, as in <code>createCanvas(900, 500, myCanvas)</code>, then it will
  be used

  by the sketch.</p>

  <p>The fourth parameter is also optional. If an existing

  <a href="https://developer.mozilla.org/en-US/docs/Web/API/HTMLCanvasElement/"
  target="_blank">HTMLCanvasElement</a>

  is passed, as in <code>createCanvas(900, 500, WEBGL, myCanvas)</code>, then it
  will be

  used by the sketch.</p>

  <p>Note: In WebGL mode, the canvas will use a WebGL2 context if it's supported

  by the browser. Check the <a
  href="/reference/p5/webglVersion/">webglVersion</a>

  system variable to check what version is being used, or call

  <code>setAttributes({ version: 1 })</code> to create a WebGL1 context.</p>
