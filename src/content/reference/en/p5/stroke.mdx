description: >
  <p>Sets the color used to draw points, lines, and the outlines of shapes.</p>

  <p>Calling <code>stroke(255, 165, 0)</code> or <code>stroke('orange')</code>
  means all shapes drawn

  after calling <code>stroke()</code> will be filled with the color orange. The
  way

  these parameters are interpreted may be changed with the

  <a href="/reference/p5/colorMode/">colorMode()</a> function.</p>

  <p>The version of <code>stroke()</code> with one parameter interprets the
  value one of

  three ways. If the parameter is a <code>Number</code>, it's interpreted as a
  grayscale

  value. If the parameter is a <code>String</code>, it's interpreted as a CSS
  color

  string. A <a href="/reference/p5/p5.Color">p5.Color</a> object can also be
  provided to

  set the stroke color.</p>

  <p>The version of <code>stroke()</code> with two parameters interprets the
  first one as a

  grayscale value. The second parameter sets the alpha (transparency) value.</p>

  <p>The version of <code>stroke()</code> with three parameters interprets them
  as RGB, HSB,

  or HSL colors, depending on the current <code>colorMode()</code>.</p>

  <p>The version of <code>stroke()</code> with four parameters interprets them
  as RGBA, HSBA,

  or HSLA colors, depending on the current <code>colorMode()</code>. The last
  parameter

  sets the alpha (transparency) value.</p>
