description: >
  <p>Resizes the image to a given width and height.</p>

  <p>The image's original aspect ratio can be kept by passing 0 for either

  <code>width</code> or <code>height</code>. For example, calling
  <code>img.resize(50, 0)</code> on an image

  that was 500 × 300 pixels will resize it to 50 × 30 pixels.</p>

      // Resize the image.
      img.resize(50, 100);

      // Display the resized image.
      image(img, 0, 0);

      describe('Two images of a mountain landscape. One copy of the image is squeezed horizontally.');
    }
