description: >
  <p>Stops creating a hole within a flat shape.</p>

  <p>The <a href="/reference/p5/beginContour">beginContour()</a> and
  <code>endContour()</code>

  functions allow for creating negative space within custom shapes that are

  flat. <a href="/reference/p5/beginContour">beginContour()</a> begins adding
  vertices

  to a negative space and <code>endContour()</code> stops adding them.

  <a href="/reference/p5/beginContour">beginContour()</a> and
  <code>endContour()</code> must be

  called between <a href="/reference/p5/beginShape">beginShape()</a> and

  <a href="/reference/p5/endShape">endShape()</a>.</p>

  <p>Transformations such as <a href="/reference/p5/translate">translate()</a>,

  <a href="/reference/p5/rotate">rotate()</a>, and <a
  href="/reference/p5/scale">scale()</a>

  don't work between <a href="/reference/p5/beginContour">beginContour()</a> and

  <code>endContour()</code>. It's also not possible to use other shapes, such as

  <a href="/reference/p5/ellipse">ellipse()</a> or <a
  href="/reference/p5/rect">rect()</a>,

  between <a href="/reference/p5/beginContour">beginContour()</a> and
  <code>endContour()</code>.</p>

  <p>Note: The vertices that define a negative space must "wind" in the opposite

  direction from the outer shape. First, draw vertices for the outer shape

  clockwise order. Then, draw vertices for the negative space in

  counter-clockwise order.</p>
