Saturation

Horizontal bars that step through saturation(high to low) of a color, controlled using a loop

Saturation is the strength or purity of the color and represents the amount of gray in proportion to the hue. A “saturated” color is pure and an “unsaturated” color has a large percentage of gray.

function setup() {
  createCanvas(400, 400) ;
  colorMode(HSB);
}

function draw() {
  background (220)
  noStroke();

  for (let i = 0;i<6;i++){
    
    //with each iteration of the loop 
    //the saturation steps down by 20
    
    //fill(hue,saturation,brightness)
    fill (28,100 - i*20,95) ;

    rect (0,i*height/6,width,height/6) ;
  }
}
