Accessibility

Simple Draw

Touch to draw on the screen using mouseX, mouseY, pmouseX, and pmouseY values.


function setup() {
  createCanvas(displayWidth, displayHeight);
  strokeWeight(10);
  stroke(0);
}

function touchMoved() {
  line(mouseX, mouseY, pmouseX, pmouseY);
  return false;
}
